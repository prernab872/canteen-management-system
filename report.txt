CANTEEN MANAGEMENT SYSTEM - PROJECT REPORT


##  Introduction
The Canteen Management System is a simple project developed using C programming language.  
It helps in managing food items, customer orders, and bill generation in an easy and efficient way using file handling and structures.

## This Project is Needed
Managing a canteen manually can be time-consuming and confusing.  
This project helps to:
- Reduce manual work
- Manage menu items easily
- Generate bills automatically
- Store records using files
- Improve order management

--------------------------------------------------------

#  How the Project Works
1. Admin logs into the system.
2. Admin can add or delete food items from the menu.
3. Customers can view the menu and place orders.
4. The system calculates the total bill automatically.
5. Bill details are saved in a file.

-----------------------------------------------------

#  Features
- Admin Login
- Menu Display
- Add Food Items
- Delete Food Items
- Order Handling
- Automatic Bill Generation
